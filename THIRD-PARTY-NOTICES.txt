BMZ Player Third-Party Notices
==============================

This file covers third-party notices that are not generated from Cargo crate
metadata, such as FFmpeg, the ASIO SDK, and bundled skin assets. It is not legal
advice.

Rust crate license texts are generated separately with cargo-about and are
packaged next to this file:

  resources/licenses/third-party-notices.txt
  resources/licenses/rust-dependency-licenses.txt
  Contents/Resources/licenses/third-party-notices.txt
  Contents/Resources/licenses/rust-dependency-licenses.txt

The BMZ Player in-app license panel displays this file together with the
generated Rust dependency license report when that report is present.


FFmpeg
------

BMZ Player uses FFmpeg through the Rust ffmpeg-next crate for audio sample
decoding and video BGA decoding.

FFmpeg is normally licensed under LGPL-2.1-or-later. FFmpeg builds that include
GPL components are GPL. FFmpeg builds configured with nonfree components may be
non-redistributable.

Release packages must preserve the notices and source/provenance information for
the exact FFmpeg binaries or dynamic libraries included in the artifact.

Project:
  https://ffmpeg.org/

Legal information:
  https://ffmpeg.org/legal.html


ASIO SDK
--------

BMZ Player's Windows ASIO support uses the Steinberg ASIO SDK through Rust
audio integration crates.

Steinberg's public ASIO open-source license page states that ASIO technology is
available under GPLv3. The ASIO name and logo are Steinberg trademarks; use of
the name and logo is separate from the GPL terms and must follow Steinberg's
usage guidelines.

BMZ Player uses "ASIO" only descriptively, for example "ASIO support" or
"ASIO driver". BMZ Player does not include the ASIO logo.

Project:
  https://www.steinberg.net/developers/asiosdk-open/
  https://www.steinberg.net/asiosdk


Bundled Skins
-------------

BMZ Player bundles first-party and third-party skin assets under resources/skins.

data/skins/default:
  First-party BMZ Player minimal skin, covered by the BMZ Player license.

data/skins/Rmz-skin:
  Bundled Git submodule for the BMZ Player distribution. Its files include
  GPLv3-covered materials and CC BY-NC-ND 4.0-covered materials. Preserve the
  submodule's README and _license/ notices in release packages.

data/skins/mz-select:
  Bundled Git submodule for the BMZ Player distribution. Preserve readme.txt and
  the license/ directory in release packages. Bundled fonts and VOICEVOX audio
  have separate terms documented there.
